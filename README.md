# 🎲 Dice Roll App

A simple and interactive **Dice Roll Web App** built using HTML, CSS, and JavaScript. This app simulates rolling a dice and generates a random number between **1 and 6**, displaying the result visually using dice images.

---

## 🚀 Features

* 🎲 **Random Dice Roll** – Generates a number between 1 and 6 on each click
* 🖼️ **Dynamic Image Update** – Displays the corresponding dice face
* 🖱️ **Interactive Button** – Roll the dice with a single click
* ⚡ **Instant Result** – Updates immediately without page refresh

---

## 🧠 Concepts Used

* DOM Manipulation
* Event Handling (Click Events)
* Random Number Generation
* Dynamic UI Updates

---

## 📂 Project Structure

* `HTML` → Structure of the app
* `CSS` → Styling and layout
* `JavaScript` → Logic for random dice roll
* `Assets (Pics)` → Dice images (1–6)

---

## 🎯 How It Works

1. User clicks the **Dice button**
2. A random number (1–6) is generated
3. Based on the number:

   * The corresponding dice image is selected
4. The image updates instantly on the screen
